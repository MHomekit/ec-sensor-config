Device: Not connected

WiFi Configuration

How to Configure:

Step 1: Click "Connect to EC Sensor via Bluetooth" button
Step 2: Select your EC sensor device from the Bluetooth device list
Step 3: Select your WiFi network from the list or enter manually
Step 4: Enter your WiFi password
Step 5: Click "Send Configuration to Device" button
Step 6: The device will save the configuration and connect to your WiFi

Browser Compatibility:

Web Bluetooth API is supported in:

  • Android Chrome 79+
  • Android Edge 79+
  • Chrome OS Chrome 79+
  • macOS Chrome 79+
  • Linux Chrome 79+
  • Windows Chrome 79+

Note: iOS Safari does not support Web Bluetooth API.
